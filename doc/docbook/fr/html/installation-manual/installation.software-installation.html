Installation logicielle

  • Scenic Manuel d'installation @PACKAGE_VERSION@

Une installation de Scenic standard exige au moins deux ordinateurs équipés de systèmes d’exploitation linux identiques, des librairies et paquets de développement requis ainsi que la plus récente version du logiciel Scenic.

Les ordinateurs doivent satisfaire aux exigences minimales suivantes : Pré-requis matériel.

Veuillez noter qu’avant de pouvoir utiliser Scenic, il vous fait également disposer d’un réseau répondant aux spécifications exigées. Spécifications réseau
