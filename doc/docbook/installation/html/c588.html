Chapter 5. Configure audio and video

First verify that you have all required dependencies at our Software Installation page.

5.1. Setting A/V Transmission

Then add your username to the different groups:

Add your user to the video group 

sudo gpasswd -a "username" video

Add your user to the audio group 

sudo gpasswd -a "username" audio

For firewire-based hardware (DV, IIDC, freebob/ffado) 

sudo gpasswd -a "username" disk

To be in effect, you have to log out, then log in.
