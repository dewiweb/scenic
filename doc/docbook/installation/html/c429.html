Chapter 4. Software Installation

A standard Scenic installation always requires at least two computers with identical linux operating systems, the required libraries and development packages, and a build of the most recent release of the Scenic software.

The computers must meet these minimum requirements.

Note that prior to using Scenic you also need to meet the required network specification

For The HelloWorld project partners, we strongly recommend to follow our HelloWorld Steps page where we describe how to perform a network test between two partners.

4.1. Operating System Installation

Scenic operates on the Linux based desktop operating system and is tested on the following Linux distribution:

Distribution

Status

Ubuntu 8.04 LTS (Hardy)

Fully working

Ubuntu 9.04 (Jaunty)

Not Working

Ubuntu 9.10 (Karmic)

Working, less well tested and mostly documented

Debian 5.0.4 (Lenny)

Working, not much tested and getting more documented

Debian (Squeze)

Untested

To install Ubuntu, please follow the instructions found on the Get Ubuntu page.

To install Debian, please follow the instructions found on the Get Debian page.
