Chapter 7. Scenic in command line

The use of Scenic in command line is a powerfull tool with many options not implemented yet in the Scenic GUI.

The actual name of Scenic commandline is "milhouse".

For more complex setup than what Scenic can currently support, we suggest you to use lunch as a launching script. Look at some examples in trunk/utils/

7.1. Start the sound server

The sound server is an application used to manage the soundcard. Choosing which soundcard to use, seleting the input and output and defining the audio quality. It usually runs as a background process.

Follow our Audio settings page to start the sound server.
