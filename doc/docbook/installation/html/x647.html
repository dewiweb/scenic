5.3. Audio Settings

First verify that you have an appropriate sound card on our HardwareRequirements.

5.3.1. Firewire sound card

Verify that your soundcard is detected For firewire soundcard: 

ls -l /dev/*1394*

If the device does not appear try 

sudo modprobe raw1394

The current user must be part of the disk group. If that is not the case, then the system administrator must add the "user" to the "disk" group (as shown above)

5.3.2. For PCI soundcard

For pci soundcard with alsa driver, verify that your soundcard is detected: 

aplay -l
