7.3. Start the senders

Execute program on local machine with the following parameters: 

milhouse --sender --address remote_ip_address --videocodec mpeg4 --videoport 10030[1]

Execute program on remote machine with the following parameters: 

milhouse --sender --address local_ip_address --videocodec mpeg4 --videoport 10030[2]

A video window showing live video should appear on one of your designated screens. It is movable by holding down the -alt- key and dragging the window. Pressing 'F' will toggle full screen mode.

You should also see the audio connections appear in qjackctl. The outputs are now routable to your speakers or other applications.

You should now have video and audio input fully working on both sides.

To stop milhouse type ctrl-c in the used terminal or type qq in the video screen

Notes

[1]

--videosource v4l2src --audiosource jackaudiosrc --audiocodec raw  -c 2

--audioport 10000
