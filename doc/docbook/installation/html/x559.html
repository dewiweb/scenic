4.3. Installing the midi component

The midi component in Scenic is available on the unstable branch (0.5). You can skip this if you install Scenic 0.4.x and you don't want to stream midi, even with the command line.

Install the python-portmidi package: https://bitbucket.org/aalex/pyportmidi/downloads/ Note: this package contains patches unavailable with the python bindings that comes with portmidi.
