6.4. Remote Network Testing

Perform a traceroute starting from your computer to the remote computer. This will show you the route taken by the packets across the network. You can use this to determine if the packets are traveling on the proper ISP. 

traceroute "Remote IP Address"

Now, repeat the same steps as the Local Network Test but this time between your local network and the remote network.
