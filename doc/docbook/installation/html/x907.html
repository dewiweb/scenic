6.2. Network speed

Network speed and performance are of the utmost importance. To ensure that the network can handle the Scenic setup, system administrators / users must perform the tests, described below, on the local network and between two endpoints.

The following performance rates must be met:

Bandwidth: 3.5Mbps video + 1.6Mbps audio = > 5Mbps + 20%
