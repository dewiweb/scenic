4.4. Building Scenic

To get the latest Scenic version, go to http://scenic.sat.qc.ca/download

Extract the tarball and cd into it: 

tar xzj scenic-0.4.x.tar.gz
cd scenic-0.4.x

For Debian 5.0.4 (lenny), you need to install liblame. Please skip this step for other distributions. 

./utils/install_lame.sh

To get all gstreamer tarballs and build all the gstreamer modules, from the current directory, run: 

./utils/install_gstreamer.sh

This script runs each module's autogen.sh script, followed by make and sudo make install.

Now type ldconfig command:

For Ubuntu 8.04 LTS (Hardy) you need to install boost version 1.35. Please skip this step for other distributions.

./utils/boost1_35_for_hardy.sh

Next, compile scenic: 

cd scenic
./configure && make && sudo make install
