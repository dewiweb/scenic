7.2. Start the receivers

Set the local_machine display 

export DISPLAY=:0.0

Execute program on the local machine with the following parameters: 

milhouse --receiver --address remote_ip_address --videosink xvimagesink --videocodec mpeg4 [1]

Set the remote_machine display 

export DISPLAY=:0.0

Execute program on the remote machine with the following parameters:

milhouse --receiver --address local_ip_address --videosink xvimagesink --videocodec mpeg4[2]

Notes

[1]

--videoport 10030 --audiosink jackaudiosink --audiocodec raw --audioport 10000

--numchannels 2
[2]

--videoport 10030 --audiosink jackaudiosink --audiocodec raw --audioport 10000

--numchannels 2
