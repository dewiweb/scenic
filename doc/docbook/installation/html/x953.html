6.3. Bidirectional Local Network Testing

This iperf bidirectional test evaluates network performances in the the context of bidirectional video and audio transmission.

NOTE: For now we do not recommend the -d option in iperf as it generates unwanted warnings and errors.

Single-sided bidirectional test: 

iperf -c ''server ip address'' -b 30M -r -u

The -r option does a bidirectional test individually, you should get 0% packet loss on both sides.

Simultaneous client/server test (on computers A and B): 

iperf  -s  -u

On computers A and B, started at the same time, you should get 0% packet loss on both sides. 

iperf  -c ''server ip address'' -b 30M -u
