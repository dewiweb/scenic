6.3. Network bandwidth

Network bandwidth is a measure of available or consumed data communication resources expressed in bits or multiples of it (kbps, Mbps etc). To determine the required level of needed bandwidth, you have to calculate the Mbps rate of the audio and video tracks (including compression rates). Example: a 3.5 Mbps MPEG4 video with 2 x 800kbps audio channels requires 5.1 Mbps. For best results, users should add to this a 20% leeway and also consider additional network usage such as chat, skype, web browsing and so on.
