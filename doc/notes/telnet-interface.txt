(c)ontacts all
sage: contacts [options]

Manage the address book.

Options:
  -h, --help            show this help message and exit
  -l, --list            List all the contacts
  -a ADD, --add=ADD     Add a contact
   -a name ip [port]
  -e ERASE, -d ERASE, --erase=ERASE, --delete=ERASE
                        Erase a contact
  -m MODIFY, --modify=MODIFY
                        Modify a contact
   -m oldname newname ip [port]
  -s SELECT, --select=SELECT
                        Select a contact

Ex: 
pof: contacts -a tzing 10.10.10.155 

pof: c -m tzing tzing 10.10.10.66
